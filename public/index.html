Derp-Octo-Archer

A GitHub issues editor. Connect to github to get started.

Connect

Planning

Plan milestones using a drag and drop interface. Create new issues and assign estimates.

Tasks

Use a 'card wall' to drag and drop items to new statuses with auto assigning.

Evolving, open and still GitHub

Features are constantly being added. The project is, of course, hosted on GitHub and planned using itself.

Because this is just a 'view' to the GitHub issues system, everything you do through the planning or tasks views is viewable and editable through the standard interface.

Check it out and feel free to contribute!
